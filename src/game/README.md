# Логика движка

## Базовый класс
Отвечает за отрисовку канваса и объявление объектов

## Юниты
У нас есть объект "юнит", от которого наследуются все остальные юниты. Юнитами мы считаем все движущиеся объекты (игрок, враги, снаряды)
